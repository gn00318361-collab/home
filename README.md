## 使用方式

1. 用瀏覽器開啟 `家庭教育影片推薦.html`
2. 點擊影片即可直接播放（需連接網路）
3. 播放完畢後，可參考每部影片下方的「討論引導問題」帶領親子對話

- 學校家庭教育日活動

- 親子共學活動
